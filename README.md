# Pet Care Social Network

Pet Care is a social networking web application designed for pet lovers to share content and connect with other users.

## Features
- User signup and login authentication
- Upload posts with text and images
- Like posts from other users
- Follow and unfollow other users
- Update and manage user profiles
- View posts from other users

## Technologies
- HTML
- CSS
- JavaScript
- Backend logic

## Tools
- Visual Studio Code
- Git
